# KY-Code

> VSCode/VSCodium deep customization, fused with 15 open-source project features, optimized for Chinese users.

## Features

- Fully localized, data not uploaded
- iOS-style rounded UI, modern design
- Built-in AI agent system (multi-model API Key)
- 23 functional modules (real implementation)
- GitHub auto-update check
- Government-related function templates
- Full Chinese interface, out of the box

## Download & Install

1. Go to [Releases](../../releases)
2. Download latest `KY-Code-Setup-*.exe`
3. Run installer, default to D drive
4. Login with admin account
